# 交易类型预制件库

> 每种交易类型包含完整的预制件：必问清单、分析框架、方案空间、风险清单、文书模块。
> Phase 1识别出交易类型后加载对应预制件，驱动后续全部环节。

---

## T1 · 自然人直接转让

### Phase 1 预制

**必问清单：**
1. 转让价格（约定价 or 待协商）
2. 股权原始成本 + 取得方式（出资/受让/继承/赠与→决定原值确认方法）
3. 标的公司土地房产占总资产比例（>20%→评估要求）
4. 标的公司未分配利润（决定方案空间）
5. 实缴 vs 认缴（影响印花税计税依据）
6. 双方有无关联关系（影响核定风险）

**典型缺口：** 用户通常会说转让价格和持股比例，但不主动说未分配利润、土地房产占比、认缴/实缴差异。

**初步判断模板：**
- 转让价 > 净资产份额 → 不触发核定，基线清晰
- 转让价 < 净资产份额 + 不属于67号公告第13条正当理由 → 核定风险高
- 未分配利润大 + 转让方为自然人 → ⚠ 触发TP-01检测：先分红对自然人无效（税率均20%）
- 未分配利润大 + 存在法人股东 → 法人间分红免税通道可能有效
- 土地房产>20% → 需评估报告，核定风险上升

### Phase 3 预制 · 分析

**适用税种（确定）：**
1. ✅ 个人所得税：财产转让所得20%，67号公告
2. ✅ 印花税：产权转移书据万分之五，自然人减半万分之二点五
3. ❌ 增值税：不涉及（非上市，36号文）
4. ❌ 契税：不涉及（49号公告）
5. ⚠ 土地增值税：一般不涉及，但占比高+100%转让时在Phase 3风险评估中标注

**计算模板：**
```
个税 = (转让收入 - 股权原值 - 合理费用) × 20%
  转让收入 = 约定价格（需 ≥ 净资产份额，否则核定）
  股权原值 = 按67号公告第15条确认
  合理费用 = 印花税等税费
印花税 = 合同金额 × 0.025%（自然人减半）或 0.05%
  计税依据扣除认缴未实缴部分（2022年第22号公告）
```

**方案空间：**
| 方案 | 前提条件 | 预期效果 | 已知结论 |
|------|---------|---------|---------|
| 先分红再转让 | 有未分配利润 | ⚠ **自然人场景无效**（TP-01）| 分红20%与转让20%抵消 |
| 通过持股平台 | 有已设立的合伙企业 | ⚠ 大概率更贵 | "进场"本身就是一次转让+税率争议 |
| 分期付款/延迟纳税 | 买方同意分期 | 现金流改善但总税不变 | — |
| 合理费用最大化 | 有可扣除的费用 | 微量节税 | — |

### Phase 3 预制 · 风险

**典型风险：**
- 价格核定：转让价 vs 净资产份额对比（67号公告第11-14条）
- 关联交易：如有，额外关注
- 土增税穿透：占比高时标注
- 扣缴义务：受让方为扣缴义务人（67号公告第5条）

**文书特殊模块：** 无（标准备忘录即可）

## T2 · 法人直接转让

### Phase 1 预制

**必问清单：**
1. 转让方企业的所得税税率档位（一般25%/高新15%/小微实际5%）
2. 标的公司未分配利润（法人间分红免税→先分红方案有效）
3. 标的公司盈余公积
4. 转让方是否为居民企业直接投资（分红免税的前提）
5. 转让方有无可用的亏损抵扣

**初步判断模板：**
- 未分配利润大 → **先分红方案有效**（法人间分红免税，企税法第26条）
- 高新技术企业15%税率 → 节税空间相对有限
- 有亏损可抵扣 → 转让所得可用亏损冲抵

### Phase 3 预制 · 分析

**适用税种：**
1. ✅ 企业所得税：25%/15%/5%
2. ✅ 印花税：万分之五（非小型微利不减半）或万分之二点五（小型微利减半）
3. ❌ 增值税：不涉及
4. ❌ 契税：不涉及
5. ⚠ 土地增值税：同T1

**计算模板：**
```
企税 = (转让收入 - 投资计税基础) × 税率
  ⚠ 国税函〔2010〕79号第3条：不得扣除被投资企业留存收益中应分配的金额
  → 这正是"先分红再转让"有效的法理基础
印花税 = 合同金额 × 0.05%（或0.025%小型微利）
```

**方案空间：**
| 方案 | 前提条件 | 预期效果 | 备注 |
|------|---------|---------|------|
| 先分红再转让 | 被投资企业有留存收益+居民企业直接投资 | **有效** | 分红免税，转让价降低→企税降低 |
| 利用亏损抵扣 | 转让方有未弥补亏损 | 转让所得可与亏损相抵 | 五年弥补期限 |
| 特殊性税务处理 | 符合财税〔2009〕59号 | 递延纳税 | 门槛高，需收购75%以上 |

### Phase 3 预制 · 风险

**典型风险：**
- "先分红再转让"的时间间隔风险：分红未实际划转+紧接着转让→反避税风险（争议9，TP-10）
- 特殊性税务处理的条件判断错误：85%股权支付+50%股权收购双门槛严格，误判可能导致递延纳税被否
- 高新技术企业15%税率的认定有效期：高新证书3年有效，到期未续则税率回到25%
- 亏损抵扣的五年弥补期限：超期的亏损不可用

## T3 · 通过合伙企业转让

### Phase 1 预制

**必问清单：**
1. 合伙企业是否为已备案创投企业
2. 如为创投→选择的核算方式（单一基金/年度整体）
3. 各合伙人的身份和份额（自然人LP/法人LP/GP）
4. 合伙企业注册地（地方政策差异）

**初步判断模板：**
- 创投 + 单一基金核算 → 税率20%确定
- 非创投 → ⚠ 税率争议（争议3）：主流5%-35%，少数观点20%
- 法人LP → 企税25%，无争议
- 有分红可分 → 分红穿透按20%（84号函），优于转让通道

### Phase 3 预制 · 分析

**适用税种：**
1. ✅ 个税（自然人LP穿透）：税率有争议
2. ✅ 企税（法人LP穿透）：25%
3. ✅ 印花税
4. ❌ 增值税（非上市）
5. ❌ 契税

**已知争议：** 加载 dispute-cases.md 争议3

**方案空间：**
| 方案 | 前提 | 效果 |
|------|------|------|
| 先分红再通过合伙企业转让 | 被投资企业有留存收益 | 分红穿透按20%（确定），转让通道可能35%，可锁定部分收益在低税率通道 |
| 创投备案 | 符合条件 | 20%税率通道 |

### Phase 3 预制 · 风险

**典型风险：**
- 税率争议（争议3）：非创投合伙企业股权转让所得5%-35% vs 20%，口径不确定是核心风险
- 创投企业资格认定的持续合规：创投备案条件变化可能影响税率适用
- 地方口径差异（争议8）：注册地选择影响有限，2021年后洼地清理
- 分红穿透的正确性：合伙企业收到分红后穿透到自然人LP按20%，但转付路径需清晰
- 创投8号文有效期至2027-12-31（时效性风险）

## T4 · 对赌回购

### Phase 1 预制

**必问清单：**
1. 对赌条款的具体约定（固定年化回报？业绩对赌？）
2. 回购价格计算方式（固定公式 or 可谈判）
3. 回购义务人是创始人个人还是公司
4. 投资方的身份（自然人/法人/合伙企业）

**初步判断模板：**
- 固定年化回报措辞 → 倾向利息定性
- 业绩对赌 + 浮动价格 → 倾向股权转让定性
- 必须双口径分析（争议1）

### Phase 3 预制 · 分析

**适用税种：** 取决于定性
- 口径A（股权转让）：个税/企税（视投资方身份）+ 印花税
- 口径B（利息）：利息/股息红利所得 + 可能的增值税

**已知争议：** 加载 dispute-cases.md 争议1

**方案空间：** 有限。主要是条款设计影响定性（但对赌已签，事后改动空间小）。

### Phase 3 预制 · 风险

**典型风险：**
- 定性争议本身就是风险——两种口径税负可能差异巨大
- 如回购义务人是个人→个人现金流压力

**文书特殊模块：** 争议分析section需要加重篇幅，两种口径并排呈现。

## T5 · 离婚股权分割

### Phase 1 预制

**必问清单：**
1. 协议离婚还是判决离婚
2. 股权分割方案（各自保留 or 有交叉转移）
3. 有无对价/补偿款
4. 股权是否为夫妻共同财产

**初步判断模板：**
- 各自保留各自名下股权 + 无对价 → 无税务影响
- 一方股权转移给另一方 + 无对价 → 大概率不征（参照财产分割原则），但无明文
- 有对价/补偿款 → ⚠ 可能被重新定性为有偿转让

### Phase 3 预制 · 分析

**适用税种：** 一般不涉及（无明确法条规定离婚股权分割征个税）
**文书特殊模块：** 标注"无总局层面专门文件"的不确定性。

### Phase 3 预制 · 风险

**典型风险：**
- 无总局层面专门文件的法律不确定性（争议7）
- 有对价/补偿款时被重新定性为有偿转让的风险
- 离婚协议中股权分割比例与原持股比例不一致时的核定风险
- 离婚后一方持有的股权成本基础如何确认——无明确规则

## T6 · 亲属间赠与

### Phase 1 预制

**必问清单：**
1. 赠与双方的具体亲属关系（必须核对67号公告第13条第(二)项列举范围）
2. 受赠方未来是否计划转让（影响成本基础的重要性）
3. 受赠方是否为未成年人

**亲属范围（67号公告第13条第(二)项）：** 配偶、父母、子女、祖父母、外祖父母、孙子女、外孙子女、兄弟姐妹、直接抚养/赡养义务人。**叔侄、表亲不在范围内。**

**初步判断模板：**
- 范围内亲属 → 零个税，仅印花税
- 范围外亲属 → 不享受豁免，按正常转让处理（可能被核定）
- 需同时评估未来转让时的成本基础（原持有人原值，可能很低→未来税负高）

### Phase 3 预制 · 分析

**适用税种：**
1. 个税：零（范围内亲属）
2. ✅ 印花税
3. ❌ 其他

**受赠方成本基础：** 原持有人原值 + 合理税费（67号公告第15条第(三)项）

**⚠ 与继承对比：** 两者成本基础相同，当前环节差异仅在印花税。

### Phase 3 预制 · 风险

**典型风险：**
- 亲属范围判断错误：叔侄、表亲等不在67号公告第13条范围内，误判为免核定将导致补税
- 成本基础极低的传递效应：受赠方成本=原持有人原值（可能很低），未来转让时税负高
- 赠与合同金额为零时印花税计税依据的争议
- 受赠方为未成年人时的法律行为能力问题

## T7 · 继承

### Phase 1 预制

**必问清单：**
1. 被继承人是否已去世（否则是赠与不是继承）
2. 有无遗嘱
3. 继承人人数和继承方案
4. 继承人未来是否计划转让

**初步判断模板：**
- 被继承人已去世 → 继承环节零个税+印花税多数不征
- 成本基础传递：继承人的股权原值 = 被继承人原值（通常很低→未来转让税负高）
- 多继承人 → 需确认继承方案（法定继承/遗嘱继承），各继承人的份额和成本基础分别计算
- 继承人计划近期转让 → 重点分析继承后转让的全生命周期税负，与赠与方案对比
- 继承人为未成年人 → 法定代理人代为办理，提示监护人责任

### Phase 3 预制 · 分析

**适用税种：**
1. 个税：不征（继承环节）
2. 印花税：争议（多数地区不征）
3. ❌ 其他

**继承人成本基础：** 被继承人原值 + 合理税费

### Phase 3 预制 · 风险

**典型风险：**
- 成本基础极低风险：继承人成本 = 被继承人原始出资（可能非常低），未来转让时应纳税所得额高
- 多继承人遗产分割争议延长交易时间
- 继承公证/法院确认的时间成本
- 股权继承是否需要其他股东同意（公司章程可能有约定）

## T8 · 减资退出

### Phase 1 预制

**必问清单：**
1. 退出股东的身份类型（自然人 vs 法人→处理方式完全不同）
2. 标的公司累计未分配利润
3. 标的公司累计盈余公积
4. 回购/减资价格

**初步判断模板：**
- 自然人股东减资 → 全部按"财产转让所得"20%（41号公告），**不做三段拆分**
- 法人股东减资 → 三段拆分（34号公告）：投资收回 + 股息（居民企业间免税）+ 转让所得
- 法人股东且有大额留存收益 → 三段拆分中"股息所得"部分免税是核心优化点

### Phase 3 预制 · 分析

**适用税种：**
- 自然人：个税20%（41号公告）+ 印花税
- 法人：企税（34号公告三段拆分）+ 印花税

**计算模板（法人三段拆分）：**
```
减资取回金额
  - 第一段：投资收回 = min(取回金额, 初始投资成本) → 不征税
  - 第二段：股息所得 = 累计留存收益 × 减资比例 → 居民企业间免税
  - 第三段：转让所得 = 取回金额 - 投资成本 - 股息所得 → 征企税
```

**⚠ 高频错误（TP-06）：** 不得用企税三段拆分逻辑去算个税。个税按41号公告，只有一步：(取回金额-原始出资)×20%。

### Phase 3 预制 · 风险

**典型风险：**
- TP-06：自然人减资误用企税三段拆分（最高频错误）
- 三段拆分中"减资比例"的计算争议：是按减资的注册资本比例还是按取回金额占总净资产的比例？
- 回购价格是否公允：如回购价低于净资产份额且无正当理由，可能被核定
- 减资需要经过股东会决议+工商变更+债权人公告等程序，时间较长
- 法人股东三段拆分中"累计留存收益"的确认口径：是否包含当期利润？截止到何时？

## T9 · 增资稀释

### Phase 1 预制

**必问清单：**
1. 增资方身份
2. 增资金额和增资后持股比例
3. 是否定向增资（只对部分股东增资）

**初步判断模板：**
- 一般增资→ 原股东被同比稀释，不涉及税
- 大幅定向增资→ 可能被认定为"实质转让"触发反避税审查

### Phase 3 预制 · 分析

**适用税种：** 一般不涉及
**风险：** 反避税穿透

### Phase 3 预制 · 风险

**典型风险：**
- 大幅定向增资被认定为"实质转让"：如增资后原股东比例从90%降至10%，可能被视为变相转让
- 增资价格是否公允：增资价格显著低于净资产份额→可能被认定为利益输送
- TP-02关联：增资稀释后原股东成本基础不变（不可按比例缩减），后续转让时易算错

## T10 · 股权代持（代持设立+代持还原）

### Phase 1 预制

**必问清单：**
1. 代持原因（资格限制？隐私？限制性行业？）
2. 是否有书面代持协议
3. 代持关系存续期间是否发生过分红
4. 当前目标：是代持设立、代持还原、还是代持状态下的对外转让
5. 名义股东与实际出资人之间是否有亲属关系（影响核定风险）

**初步判断模板：**
- 代持设立 → 一般不涉及税，但需关注未来还原的税务风险
- 代持还原 → ⚠ 争议4：各地口径不统一，建议事前与税务机关沟通
- 代持状态下对外出售 → 纳税义务人为名义股东，实际出资人取得转让款另需分析

### Phase 3 预制 · 分析

**适用税种：** 取决于交易类型（还原 vs 对外转让）
- 代持还原：争议——可能不征税或视为转让
- 对外转让：按T1/T2处理（视名义股东身份）

**已知争议：** 加载 dispute-cases.md 争议4、争议6

**方案空间：**
| 方案 | 前提 | 效果 |
|------|------|------|
| 先还原再转让 | 税务机关认可还原不征税 | 实际出资人直接转让，避免二次纳税 |
| 由名义股东直接转让 | 名义股东代为交易 | 操作简单，但可能存在名义股东→实际出资人的二次纳税风险 |

### Phase 3 预制 · 风险

**典型风险：**
- 代持还原被视为转让的风险
- 名义股东纳税后实际出资人取得款项的二次纳税风险
- 代持协议效力不被税务机关认可的风险
- 代持期间分红的纳税义务归属不明风险

### 边界条件

- 代持涉及外籍个人或境外主体 → 不覆盖，提示跨境税务
- 代持涉及上市公司股票（限售股代持） → 适用39号公告的特殊规则，本Skill不深度覆盖
- 代持协议被法院认定无效 → 超出税务分析范围，提示"代持法律效力需另行评估"
- 名义股东跑路/失联 → 非税务问题，提示"股权确权需通过司法途径解决"

## T11 · 非货币资产出资入股

### Phase 1 预制

**必问清单：**
1. 出资人身份（自然人 vs 法人）
2. 出资资产类型（股权？不动产？技术？其他？）
3. 资产的原值/计税基础
4. 资产评估后的公允价值
5. 是否有现金补价
6. 出资后持股比例（法人出资→是否可能满足特殊性税务处理条件）

**初步判断模板：**
- 自然人出资 → 按财产转让所得20%缴个税，可5年内分期（41号文2015）
- 法人出资 → 视同销售确认企税，可5年分期（116号文）；或满足条件走特殊性税务处理（59号文）
- 以不动产出资 → 额外涉及增值税、契税、土增税
- 以股权出资 → 不涉及增值税（非上市），但需关注出资标的公司的土地房产占比

### Phase 3 预制 · 分析

**适用税种（自然人以股权出资）：**
1. ✅ 个税：财产转让所得20%（41号文2015）
2. ✅ 印花税：产权转移书据万分之五/减半
3. ❌ 增值税：不涉及（非上市股权）
4. ❌ 契税：不涉及

**计算模板（自然人）：**
```
个税 = (非货币资产公允价值 - 资产原值 - 合理税费) × 20%
可分期：5年内分期缴纳（备案制）
印花税 = 出资协议金额 × 0.025%（自然人减半）或 0.05%
```

**适用税种（法人以股权出资）：**
1. ✅ 企税：(公允价值 - 计税基础) × 税率，可5年分期或走特殊性税务处理
2. ✅ 印花税

### Phase 3 预制 · 风险

**典型风险：**
- 公允价值评估争议：评估价值过高→多交税，评估价值过低→可能被核定
- 以不动产出资时关联税种叠加：增值税+土增税+契税可能使整体税负远超预期
- 分期缴纳期间转让股权：现金部分应优先缴纳未缴税款（41号文2015第四条）
- 法人走特殊性税务处理的条件判断：85%股权支付+50%股权收购双门槛严格
- TP-11反模式：混用个人和法人法条

### 边界条件

- 出资资产为境外资产 → 不覆盖，提示跨境税务
- 出资资产为上市公司股票 → 提示"适用不同规则"
- 涉及不动产出资 → Phase 3覆盖股权税务部分，提示关联税种（增值税/土增税/契税）
- 评估报告缺失 → 公允价值无法确认，标注"待确认"

## T12 · 资本公积转增股本

### Phase 1 预制

**必问清单：**
1. 被投资企业的组织形式（有限责任公司 vs 股份有限公司）
2. 转增来源（股本/资本溢价？资产评估增值？盈余公积？未分配利润？）
3. 转增金额
4. 股东构成（自然人 vs 法人）
5. 企业是否为中小高新技术企业（影响分期缴纳资格）

**初步判断模板：**
- 股份有限公司 + 股票溢价发行形成的资本公积转增 → **不征个税**（198号文+289号文）
- 有限责任公司 + 资本溢价转增 → ⚠ **争议5**，主流征税
- 盈余公积/未分配利润转增 → **确定征个税**（198号文第二条：盈余公积金派发红股属于股息红利性质分配，征个税）
- 法人股东 → 股权溢价资本公积转增：不确认收入也不增加计税基础（79号函第四条）

### Phase 3 预制 · 分析

**适用税种（自然人股东，征税情形）：**
1. ✅ 个税：利息、股息、红利所得20%
2. 中小高新技术企业可分期5年

**已知争议：** 加载 dispute-cases.md 争议5

**成本基础影响：** 如已缴税，新增股本的原值 = 转增额 + 相关税费（67号公告第15条第（四）项）

### Phase 3 预制 · 风险

**典型风险：**
- TP-07反模式：不分来源一律说不征税
- 有限责任公司资本溢价转增的征税争议（争议5）导致不确定性
- 中小高新技术企业资格认定的动态性：高新证书有效期3年，过期未续可能影响分期缴纳资格
- 法人股东的成本基础影响：股权溢价资本公积转增不增加法人股东计税基础（79号函第四条），未来转让时税负更高
- 未缴税的转增：如转增时应征税但未征（如有限公司资本溢价转增），未来转让时成本基础不含转增额，税负更高

### 边界条件

- 上市公司/新三板挂牌公司转增 → 不覆盖（适用股息红利差别化政策，规则不同）
- 转增来源混合（如同时含资本溢价和盈余公积） → 需拆分分别分析，不可笼统处理
- 外籍个人股东 → 不覆盖，提示跨境税务

## T13 · 股权激励行权

### Phase 1 预制

**必问清单：**
1. 被授予企业是上市还是非上市（本Skill仅覆盖非上市）
2. 激励类型（期权？限制性股权？股权奖励？）
3. 行权价/授予价
4. 行权时/解锁时股权公允价值
5. 是否已向税务机关备案递延纳税
6. 员工是否为技术骨干或高管（递延纳税条件之一）

**初步判断模板：**
- 未选择递延纳税 → 行权时按工资薪金所得3%-45%，行权价为后续转让成本基础
- 已选择递延纳税 → 行权时暂不纳税，转让时按财产转让所得20%，成本为实际支付价款
- ⚠ 递延纳税优惠显著（最高45%→20%），但条件严格，需确认已备案

### Phase 3 预制 · 分析

**适用税种（非上市、未递延）：**
1. ✅ 个税（行权环节）：综合所得/工资薪金3%-45%
2. ✅ 个税（转让环节）：财产转让所得20%
3. ✅ 印花税（转让环节）

**适用税种（非上市、已递延）：**
1. ❌ 个税（行权环节）：暂不纳税
2. ✅ 个税（转让环节）：财产转让所得20%，税基=转让收入-取得成本
3. ✅ 印花税（转让环节）

**方案空间：** 有限。核心选择是行权时是否已选择递延纳税（事后无法补选）。

### Phase 3 预制 · 风险

**典型风险：**
- TP-08反模式：未逐项核实递延纳税7个条件
- 通过持股平台实施的股权激励不符合递延纳税条件（101号文要求本公司直接股权）
- 递延纳税期间公司情况变化：不再符合第4-6项条件时须于次月15日内补缴
- 未备案即享受递延的合规风险：未办理备案手续的不得享受递延纳税优惠
- 行权时公允价值确认争议：非上市公司按净资产法/类比法/其他合理方法确定，存在争议空间

### 边界条件

- 上市公司股权激励 → 不覆盖（适用101号文第二条+财税〔2005〕35号体系）
- 通过持股平台间接持股的激励 → 不符合递延纳税条件，按一般规则处理并提示
- 激励对象为外籍个人 → 不覆盖，提示跨境税务
- 股权奖励涉及负面清单行业 → 不得享受递延纳税，需核查行业目录

## 执行路线图通用模板

> 按纳税主体分类。各T类型引用对应模板，仅列出该类型的特殊步骤。

### EX-A · 自然人股权转让通用执行步骤（适用T1/T3自然人LP/T4/T5/T6/T10/T11自然人/T13）

**纳税义务发生时点（67号公告第20条，以下六种情形任一发生即触发）：**
1. 受让方已支付或部分支付股权转让价款
2. 股权转让协议已签订生效
3. 受让方已经实际履行股东职责或享受股东权益
4. 国家有关部门判决、登记或公告生效
5. 以股权对外投资/抵偿债务/其他非货币性交易已完成
6. 税务机关认定的其他股权已发生转移的情形

**申报时限：** 上述情形发生后的**次月15日内**

**纳税义务人与扣缴义务人：**
- 纳税人：转让方（自然人）
- 扣缴义务人：受让方
- 主管税务机关：被投资企业所在地税务机关（第19条）

**申报材料（67号公告第21条）：**
1. 股权转让合同（协议）
2. 股权转让双方身份证明
3. 按规定需要进行资产评估的（土地房产>20%），提供具有法定资质的中介机构出具的资产评估报告
4. 计税依据明显偏低但有正当理由的证明材料
5. 主管税务机关要求报送的其他材料

**被投资企业义务（67号公告第22条）：**
- 董事会或股东会结束后**5个工作日内**报送决议、会议纪要等资料
- 股东变动后**次月15日内**报送含变动信息的《个人所得税基础信息表（A表）》及变更情况说明

**扣缴义务人义务（67号公告第6条）：**
- 协议签订后**5个工作日内**将股权转让有关情况报告主管税务机关

**工商与税务的先后顺序：**
- 实务中税务申报应在工商变更登记之前或同步完成
- 67号公告未强制要求"先税后工商"，但实务中多数地区的市场监管部门要求先完成税务申报

### EX-B · 法人股权转让通用执行步骤（适用T2/T8法人/T11法人/T12法人股东）

**纳税方式：** 并入法人企业的应纳税所得额，在企业所得税汇算清缴时申报
**申报时限：** 季度预缴+年度汇算清缴（次年5月31日前）
**扣缴：** 无代扣代缴，企业自行申报

### EX-C · 非货币资产出资分期缴纳备案步骤

**适用：** T11（自然人以非货币资产出资，选择分期缴纳）

**备案时限：** 取得被投资企业股权之日的**次月15日内**（20号公告第八条）

**备案材料：**
- 《非货币性资产投资分期缴纳个人所得税备案表》
- 非货币性资产评估报告或公允价格证明
- 投资协议
- 被投资企业股权登记证明

**分期规则：**
- 自发生应税行为之日起不超过**5个公历年度内**分期缴纳
- 取得现金补价的，现金优先用于缴税
- 分期缴税期间转让股权的，应于转让之日的**次月15日内**申报缴纳
- 变更分期计划的，重新报送备案表

**主管税务机关：**
- 以不动产投资 → 不动产所在地
- 以股权投资 → 该股权对应企业所在地
- 以其他非货币资产 → 被投资企业所在地

### EX-D · 股权激励递延纳税备案步骤

**适用：** T13（非上市公司，选择递延纳税）

**备案时限：** 行权/解禁/获得之**次月15日内**（62号公告第五条）

**备案材料：**
- 《非上市公司股权激励个人所得税递延纳税备案表》
- 股权激励计划
- 董事会或股东大会决议
- 激励对象任职或技术工作情况说明
- 实施股权奖励的，另需本企业及奖励股权标的企业上一纳税年度主营业务收入构成情况说明

**年度报告义务：** 递延纳税期间，扣缴义务人应在每个纳税年度终了后**30日内**报送《个人所得税递延纳税情况年度报告表》

**未备案的后果：** 不得享受递延纳税优惠，应按一般规则（工资薪金3%-45%）计算缴纳

### EX-E · 转增股本分期缴纳备案步骤

**适用：** T12（中小高新技术企业向个人股东转增股本，选择分期缴纳）

**备案时限：** 转增股本的**次月15日内**（80号公告第三条）

**备案材料：**
- 高新技术企业认定证书（原件审核后退还，复印件留存）
- 股东大会或董事会决议
- 《个人所得税分期缴纳备案表（转增股本）》
- 上年度及转增股本当月企业财务报表
- 转增股本有关情况说明

**分期规则：** 不超过**5个公历年度内**分期缴纳

### EX-F · 特殊性税务处理报告步骤

**适用：** T2/T11法人（符合59号文+109号文条件，选择特殊性税务处理）

**报告时限：** 重组业务完成当年企业所得税年度申报时（48号公告第四条）

**报告材料：**
- 《企业重组所得税特殊性税务处理报告表及附表》
- 重组合同/协议
- 股权/资产的公允价值证明
- 当事各方的股权/资产变动情况说明
- 合理商业目的说明

**后续义务：**
- 完成后的下一年度企业所得税年度申报时，提交书面情况说明，证明12个月内条件未变
- 转让或处置重组资产/股权时，在年度纳税申报中进行专项说明
